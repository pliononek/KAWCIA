# Kalorie + Trening (PWA)

Pojedynczy plik `index.html` — PWA do liczenia kalorii (Gemini/OpenAI/DeepSeek/Claude),
zdjęć posiłków, skanowania kodów EAN (Open Food Facts) oraz pełnego modułu treningowego
(60+ ćwiczeń w bazie, stoper, timer przerwy, rekordy osobiste).

## Uruchomienie

Otwórz `index.html` w Safari na iPhone albo wrzuć na dowolny statyczny hosting
(GitHub Pages, Netlify, Cloudflare Pages, S3). Dodaj do ekranu głównego —
działa jako PWA, dane w `localStorage`.

## Konfiguracja

Przy pierwszym uruchomieniu: wpisz imię → cel kaloryczny → klucz API wybranego
providera (Gemini 1.5 Flash / GPT-4o-mini / DeepSeek / Claude Haiku 4.5).
